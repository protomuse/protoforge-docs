# ProtoForge Python SDK

The official Python SDK provides a typed client for the ProtoForge API with built-in error handling, pagination support, and type hints for all request and response models.

## Installation

```bash
pip install protoforge
```

## Quick Start

```python
from protoforge import ProtoForge

pf = ProtoForge(api_key="pk_test_your_api_key")

# Search for tracks
tracks = pf.tracks.search(genre="electronic", bpm_min=120, territory="GB")

print(f"Found {tracks.pagination.total} tracks")
print(tracks.data[0].title)  # "Neon Pulse"
```

## Common Operations

### Search Tracks

<CodeGroup>
```python SDK
tracks = pf.tracks.search(
    genre="cinematic",
    mood="dramatic",
    territory="US",
    per_page=50,
)

for track in tracks.data:
    print(f"{track.title} ({track.bpm} BPM) — {track.status}")
```

```python requests
import requests

response = requests.get(
    "https://api.protoforge.com/v1/tracks/search",
    params={"genre": "cinematic", "mood": "dramatic", "territory": "US", "per_page": 50},
    headers={"Authorization": "Bearer pk_test_your_api_key"},
)
tracks = response.json()
```
</CodeGroup>

**Response:**

```json
{
  "data": [
    {
      "id": "trk_def456",
      "title": "Cinematic Horizons",
      "bpm": 90,
      "genre": "cinematic",
      "status": "available"
    }
  ],
  "pagination": { "page": 1, "per_page": 50, "total": 23 }
}
```

### Create a License

<CodeGroup>
```python SDK
license = pf.licenses.create(
    track_id="trk_def456",
    usage_type="film",
    territory=["US", "GB"],
    duration_months=24,
    exclusive=False,
    notes="Documentary background music",
)

print(f"License {license.id} — status: {license.status}")
```

```python requests
import requests

response = requests.post(
    "https://api.protoforge.com/v1/licenses",
    headers={
        "Authorization": "Bearer pk_test_your_api_key",
        "Content-Type": "application/json",
    },
    json={
        "track_id": "trk_def456",
        "usage_type": "film",
        "territory": ["US", "GB"],
        "duration_months": 24,
        "exclusive": False,
    },
)
license = response.json()
```
</CodeGroup>

**Response:**

```json
{
  "id": "lic_xyz789",
  "track_id": "trk_def456",
  "usage_type": "film",
  "territory": ["US", "GB"],
  "duration_months": 24,
  "status": "draft"
}
```

### List Licenses

<CodeGroup>
```python SDK
licenses = pf.licenses.list(status="active", per_page=100)
print(f"{licenses.pagination.total} active licenses")
```

```python requests
import requests

response = requests.get(
    "https://api.protoforge.com/v1/licenses",
    params={"status": "active", "per_page": 100},
    headers={"Authorization": "Bearer pk_test_your_api_key"},
)
licenses = response.json()
```
</CodeGroup>

**Response:**

```json
{
  "data": [
    { "id": "lic_xyz789", "track_id": "trk_def456", "status": "active" }
  ],
  "pagination": { "page": 1, "per_page": 100, "total": 1 }
}
```

## Error Handling

```python
from protoforge import ProtoForge, ProtoForgeError

pf = ProtoForge(api_key="pk_test_your_api_key")

try:
    license = pf.licenses.create(
        track_id="trk_invalid",
        usage_type="film",
        territory=["GB"],
        duration_months=12,
    )
except ProtoForgeError as e:
    print(f"{e.code}: {e.message}")
    # "not_found: Track trk_invalid does not exist"

    if e.details:
        for detail in e.details:
            print(f"  {detail['field']}: {detail['issue']}")
```

## Configuration

```python
pf = ProtoForge(
    api_key="pk_test_...",
    base_url="https://api.protoforge.com/v1",  # default
    timeout=30,  # request timeout in seconds (default: 30)
    retries=2,  # auto-retry on 429/5xx (default: 2)
)
```
